Boceto de Aplicación para Generación de Documentos en la Intranet de Iberdrola

Desarrollé el boceto inicial de una aplicación para la intranet de Iberdrola, destinada a automatizar la generación de documentos .docx a partir de formularios personalizados. Utilicé Express.js, Node.js y la librería docx JavaScript para crear un prototipo funcional que permitía generar más de 100 variaciones de documentos de manera automática, según los datos ingresados en los formularios.

Este boceto sirvió como base para el desarrollo final, llevado a cabo por una empresa subcontratada por Iberdrola. Mi trabajo estableció la estructura técnica y el flujo de trabajo, integrando tecnologías de frontend como HTML, CSS, JavaScript y Tailwind CSS, para proporcionar una interfaz intuitiva y responsiva, mejorando así la eficiencia en la creación de documentación interna.

Sistema de Diagnóstico de Radiografías con Inteligencia Artificial

Este proyecto de fin de curso es una aplicación web avanzada que utiliza técnicas de aprendizaje automático para el análisis de imágenes de radiografías. Desarrollado con TensorFlow en el entorno de desarrollo PyCharm y Jupyter-Lab, el sistema está construido en Python y hace uso de una base de datos de imágenes de radiografías disponible en Kaggle.


Características Principales:

Análisis Automatizado: La aplicación emplea un modelo de inteligencia artificial entrenado para identificar fracturas en radiografías. Al cargar una imagen de radiografía, el sistema analiza y proporciona un diagnóstico sobre si el hueso está roto o no.

Interfaz de Usuario Completa: La plataforma incluye funcionalidades de registro, inicio de sesión y recuperación de contraseña, permitiendo a los usuarios gestionar su acceso y seguridad de forma eficiente.

Historial de Radiografías: Los usuarios pueden consultar un historial completo de sus radiografías analizadas, gracias a la integración de bases de datos y APIs que almacenan y gestionan las imágenes de cada usuario de manera segura.


Tecnologías Utilizadas:

TensorFlow: Para el desarrollo y entrenamiento del modelo de IA que realiza el análisis de las imágenes. Python: Lenguaje principal utilizado para el desarrollo del modelo y la lógica del backend.

Jupyter-Lab: Entorno de desarrollo para la experimentación y visualización de datos.

Bases de Datos y APIs: Para el almacenamiento y gestión de imágenes y datos de usuarios.
